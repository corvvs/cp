# 問題: ABC127 D - Integer Cards

https://atcoder.jp/contests/abc127/tasks/abc127_d

# メモ

書き換える枚数は自由に選んでよい。\
また一度書き換えたものを再度書き換える意味はないので、各カードは1回書き換えるor書き換えない。\
書き換えによる利益は

- 元の数字がなるべく小さいカードを
- なるべく大きな数字に書き換える

ほうが大きくなるので、`A[i]` が小さいカードから順に、大きな`C[j]`に書き換えていく(書き換えで利益が得られなくなったら終了)、でよさそう。



# 感想

本番で3回もTLEしてしまった。反省。
