# メモ

`comb(n,r)`について:

- `n`を固定すると、`comb(n,r)`を最大にする`r`は
  - `r = n/2` (`n`が偶数)
  - `r = (n+1)/2 and (n-1)/2` (`n`が奇数)
- `r >= 0`のとき、`comb(n+1,r) = comb(n,r) * (n+1) / (n-r+1) >= comb(n,r)`
  - 等号は`r = 0`のときのみ

が成り立つ。

これより、`m = max{ a[] }`とすると、\
任意の`i`について`comb(m,r) >= comb(a[i],r)`となるので、`n = m`として損がない。\
あとは、`a[]`のうち`n/2`に最も近い(`m`以外の)要素を`r`にすればよい。\
計算量は`O(N)`.
