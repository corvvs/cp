# メモ

(解説にも書いてあるので手短に)

数列`a[]`が昇順ソートされているとする。\
`A == 1`の場合は`a[]`がそのまま答えになるので、以後`A > 1`とする。\
この時、`a[0] * A >= a[N-1]`が成り立っているとすると、それ以降は「`a[]`の先頭が最後尾に回る」という動きを繰り返すだけになる。\
よって、↑が十分小さい操作回数で成り立つことがわかれば万々歳なのだが、\
概算してみるとだいたい`N * log2(max(a[]) / min(a[])) < 1500`回やれば十分であることがわかる。
